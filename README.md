# FinSight — Team Gardenia

**FinSight** is an **AI-powered financial resilience app** designed to help Malaysian households understand their spending, prepare for unexpected financial shocks, and build stronger financial habits. By transforming everyday financial data into **clear insights and actionable guidance**, FinSight empowers users to make smarter financial decisions through a simple, mobile-first experience.

FinSight aims to promote **financial literacy, resilience, and preparedness** in an increasingly uncertain world.

---

## ✨ Key Features

### 📊 Smart Transaction Tracking
Automatically log and categorise income and expenses to help users understand where their money goes.

### 📈 Spending Insights
Visualise spending patterns with category breakdowns and intuitive charts.

### 🛡 Financial Resilience Score
Evaluate financial preparedness based on savings buffer, spending stability, and debt ratio.

### ⚡ Shock Scenario Simulator
Stress-test finances against real-life events such as job loss, illness, natural disasters, or economic crises.

### 📑 Contingency Planning
Generate personalised financial survival plans based on the user's real financial data.

### 🤖 AI Financial Educator
Receive personalised financial insights and educational micro-lessons generated by FinSight’s AI assistant.

### 📷 Receipt Scanning
Capture receipts using the phone camera to automatically record expenses.

### 🔐 Secure Authentication
JWT-based authentication with optional **Google Sign-In** support.

## 🛠 Tech Stack

| Layer | Technology |
|------|-----------|

## Prerequisites

- Python 3.11+
- Node.js 18+ and npm (or pnpm)
- [Expo Go](https://expo.dev/go) app on your phone (for quick testing), **or** Android Studio / Xcode for emulators
- A Gemini API key (get one free at [Google AI Studio](https://aistudio.google.com/))

## Backend Setup

```bash
cd backend

# 1. Create and activate a virtual environment
python -m venv .venv

# macOS / Linux
source .venv/bin/activate

# Windows PowerShell
.venv\Scripts\Activate.ps1

# 2. Install Python dependencies
pip install -r requirements.txt

# 3. Configure environment variables
cp .env.example .env
# Open .env and fill in:
#   SECRET_KEY        — any long random string
#   GEMINI_API_KEY    — from Google AI Studio
#   DATABASE_URL      — defaults to SQLite (sqlite+aiosqlite:///./dev.db)
#   TAVILY_API_KEY    — optional, for web-search grounding in AI responses

# 4. (Optional) Seed the database with mock transactions
python seed_mock_transactions.py

# 5. Start the development server
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Interactive API docs are available at:

| URL | Description |
|---|---|
| http://localhost:8000/docs | Swagger UI |
| http://localhost:8000/redoc | ReDoc |
| http://localhost:8000/api/v1/health | Health check |

## Frontend Setup

```bash
cd frontend

# Install dependencies
npm install        # or: pnpm install

# Start the Metro bundler
npm start          # or: pnpm start
```

Once Metro is running:

| Target | How |
|---|---|
| Physical device | Scan the QR code with the **Expo Go** app |
| Android emulator | Press `a` (Android Studio AVD must be running) |
| iOS simulator | Press `i` (macOS + Xcode required) |
| Web browser | Press `w` |

### Pointing the frontend at the backend

Edit `frontend/services/api.ts` (or the relevant service file) and set the base URL:

| Environment | URL |
|---|---|
| Expo web / same machine | `http://localhost:8000` |
| Android Emulator (AVD) | `http://10.0.2.2:8000` |
| Physical device (LAN) | `http://<your-machine-LAN-IP>:8000` |

## Environment Variables Reference

| Variable | Required | Default | Description |
|---|---|---|---|
| `SECRET_KEY` | Yes | `change-me-in-production` | JWT signing secret |
| `GEMINI_API_KEY` | Yes | — | Google Gemini API key |
| `DATABASE_URL` | No | `sqlite+aiosqlite:///./dev.db` | Async DB connection string |
| `ALLOWED_ORIGINS` | No | localhost ports | Comma-separated CORS origins |
| `EMAIL_ADDRESS` | No | — | Sender address for email notifications |
| `EMAIL_PASSWORD` | No | — | App password for email sender |
| `GEMINI_MODEL` | No | `gemini-2.5-flash` | Gemini model ID |
